Broadcast domain

A broadcast domain is a collection of computers in which, when one sender sends a message, the message is delivered to all others in the same domain.

routers separate them

Collision domain

A collision domain is a collection of computers in which, if more than one sender tries to send some data simultaneously, the signal will collide in the transmission media and make all the sent information unusable.

switch separate them (or hub)
