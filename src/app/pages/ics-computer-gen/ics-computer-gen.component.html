Computer Generations

1st - 1944-55

  • vaccum tubes (termion valves)
  • slow
  • power hungry
  • big and heavy
  • costly
  • hard wired : Hardwire is a function or device that is physically built into the computer instead of programmed into the software.
  • ex: ENIAC, UNIVAC, EDSAC

2nd - 1955-64

  • transistors
  • batch processing
  • fast and smaller
  • less power hungry
  • assembly language and introduction to high level language (fortran, cobol)
  • magnetic core memory
  • punch cards and tapes
  • ex: IMB 1401,IMB 1401

3rd - 1964-71

  • OS
  • high level languages
  • magnetic disk
  • mini computers
  • multi programming
  • ex: IBM System/360,IBM System/370

4th - 1971-present

  • personal computers
  • gui
  • unix
  • networking
  • parallel programming - threads
